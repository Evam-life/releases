# Incident Command Configuration Tool changelog

`````{warning}
You are viewing the Alpha releases of the Incident Command Configuration Tool.
`````

## 1.0.0

2025-06-04

### Highlights

* **Added the option to select between metric and imperial units:** It is now possible to decide which unit system is used within the Incident Command apps in Vehicle Services and Central Services.
* **Added a setting to define a custom unit within the app:** It is now possible to define a custom unit alongside its conversion rate to meters, this can be typically used for region-specific units such as Hose Lengths (25m).

## 0.9.0

2025-05-08

### Highlights

* **Initial release:** This is the initial release of this tool, bringing support for the base functionality: library management for markers and cordons, custom naming and grouping.
